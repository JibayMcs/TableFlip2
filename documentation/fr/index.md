# Bienvenue dans TableFlip

TableFlip est un studio de base de données auto-hébergé pour MySQL,
MariaDB, PostgreSQL, SQL Server et SQLite. Il fournit un explorateur
de bases dans le navigateur, un éditeur SQL, l'édition des données, des
exports et un diagramme entité-relation, derrière une page de
connexion unique.

Cette documentation décrit les fonctionnalités disponibles dans la
version actuelle. Sélectionnez une section dans la barre latérale, ou
commencez par la page [Premiers pas](/docs/getting-started) lors d'une
première visite.

## Aperçu des sections

- **Premiers pas** : les deux modes de connexion et quand choisir l'un
  ou l'autre
- **Connexions** : déclarer des bases (mode compte) ou se connecter
  directement à un serveur
- **Explorateur** : naviguer dans les schémas, consulter les lignes,
  éditer les données
- **Éditeur SQL** : exécuter des requêtes, consulter l'historique et
  utiliser le panneau SQL intégré à la vue des données
- **Exports** : générer des fichiers CSV, JSON ou des dumps SQL, avec
  compression et templates de nom
- **Diagramme** : afficher la vue entité-relation interactive de la
  base sélectionnée
- **Auto-hébergement** : faire tourner TableFlip vous-même : stack
  Docker, variables d'environnement, configuration, mises à jour et
  dépannage

Si vous opérez TableFlip pour d'autres utilisateurs (déploiement,
mises à jour), la section **Auto-hébergement** couvre ce périmètre.
