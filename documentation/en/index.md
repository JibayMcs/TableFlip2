# Welcome to TableFlip

TableFlip is a self-hosted database studio for MySQL, MariaDB,
PostgreSQL, SQL Server and SQLite. It provides a browser-based
explorer, an SQL editor, table-data editing, exports and an
entity-relationship diagram, behind a single login page.

This documentation describes the features available in the current
release. Select a section in the left sidebar to start, or read the
[Getting started](/docs/getting-started) page if this is your first
visit.

## Sections at a glance

- **Getting started** — the two ways to sign in, and when to pick each
- **Connections** — declaring databases (account mode) or connecting
  directly to a server
- **Explorer** — browsing schemas, viewing rows, editing data
- **SQL editor** — running queries, history, and the embedded scratch
  pad inside the data view
- **Exports** — CSV, JSON, and SQL dumps, with compression and
  filename templates
- **Diagram** — an interactive entity-relationship view of the
  selected database
- **Self-hosting** — running TableFlip yourself : Docker stack,
  environment variables, configuration, upgrades and troubleshooting

If you operate TableFlip for others (deployment, upgrades), the
**Self-hosting** section covers that scope.
